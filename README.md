# BELIFT - Digital Solutions

## Project Overview
BELIFT is a digital solutions provider that focuses on creating impactful online experiences for businesses. The project includes a responsive website composed of several pages: Home, About, Services, and Contact Us. This website aims to present BELIFT's mission, vision, and the services offered, providing a seamless user experience supported by modern web technologies.

## Installation
To set up the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/belift.git
   cd belift
   ```

2. **Open the project in a web browser:**
   - Simply open the `index.html` file from the project's root directory.

No server setup is necessary as this is a static HTML website.

## Usage
Visit the following pages in your web browser to explore the features of the website:

- **Home Page:** Displays the main offerings and navigation.
- **About Page:** Contains information about BELIFT's history, mission, and team.
- **Services Page:** Lists various digital services offered by BELIFT.
- **Contact Page:** Allows users to send messages and provides contact information.

## Features
- **Responsive Design:** The website is designed to be mobile-friendly, utilizing Tailwind CSS for quick styling.
- **Multi-Page Layout:** Structured across several pages to detail various aspects of the business.
- **Interactive Elements:** Includes buttons and form submissions for user engagement.
- **Professional Aesthetic:** Utilizes modern design principles to create an attractive interface.

## Dependencies
This project relies on the following external libraries:
- [Tailwind CSS](https://tailwindcss.com/)
- [Font Awesome](https://fontawesome.com/)

Both stylesheets are linked directly via CDN in the HTML files.

## Project Structure
Here's a breakdown of the project files:

```
/belift
│
├── index.html          # Home page
├── about.html          # About page
├── services.html       # Services offered
└── contact.html        # Contact form and information
```

### File Descriptions
- **index.html:** The landing page featuring an introduction to BELIFT.
- **about.html:** Contains the company's story, mission, and team profiles.
- **services.html:** Each service offered by BELIFT is described in detail with links for more information.
- **contact.html:** A form for users to get in touch along with contact details and location.
